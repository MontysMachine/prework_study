HTML Notes

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user
  • !Doctype html is a declaration to the browser to expect HTML on the page
  • title element defines the title for the webpage which is super important for search engine optimization
  • style element defines CSS styles associated to the webpage
  • base element defines base URL for the webpage
  • link element connects an external resource to HTML document
  • meta element defines metadata like char set, description, keywords, author, and viewport
  • script element embeds client side scripts into the HTML document
  • charset sets the character encoding
  • http-equiv sets the http header for content
  • viewport sets the default view for the webpage
  • div element creates a divisin in the page
  • span creates an inline division in the page
  • h1 is a heading. the bigger the number the smaller the text (ex. h1,h2,h3...)
  • p element represent a paragraph or block of text
  • ul element is an unordered list
  • ol element is an ordered list
  • li elements are list items
  • img element contains information about images displayed on the webpage
  • src attribute defines a location of an image
  • br element creates a line of empty space
  • a element defines a hyperlink typically used with href which indicates the links destination
  • alt contains a descriptive string
  • footer element appears on the bottom of the page
  • section element defines and breaks your webpage into sections
  • id attribute identifies elements and selects them

CSS Notes

  • Inline CSS when you want to style and element directly in the HTML file
  • Internal CSS uses style element in the head element.
  • External CSS is the most common way to style because it allows you to keep all CSS rules in a separate file. Use a link element in the head element
  • A selector defines an element or attribute to which rules or declarations apply
  • Declarations contain two components: the CSS property and the value of that property
  • Margin is a property that indicates how much space outside of an element
  • Padding adds space around the content inside an element
  • * is a selector that declares all element will adhere to the style properties

Git Notes

  • git is the prefix that lets your terminal know you are using a git command
  • git status shows what branch you're on and if it is up to date
  • git checkout -b (enter branch name) creates a new branch
  • add command adds modification to in the current working branch to the staging area
  • -A adds all changes
  • pull command receives a branch modification into the local enviroment
  • origin command indicates the source of the pull in the GitHub repository
  • main indicates the branch
  • push pushes changes to a remote Github branch (can create an new remote branch or add new changes to existing branch
